# USERS

Learners exploring low-level digital logic + computer architecture. Self-paced. Self-directed.

## Shapes

- Build intuition for MIPS instr via stepping through datapath animation
- Compare two instructions side-by-side
- Practice K-map grouping, self-check via solver
- Explore 5-var K-map wraparound (3D headline win)
- Derive Control unit truth table in K-map, watch same signals fire in datapath (cross-link)

## Anonymous-first

Default state. Every feature works no identity:
- Pick instr, step, run, reset
- Author truth table, group, see PIs
- Save → content-addressed permalink
- Share permalink → recipient opens same state, no wall
- Compare, pipeline, critical path — all anon

## Signed-in

Optional accent for cross-device persist:
- "My saves" across devices
- Anon snapshots in localStorage claim silently on signin
- Zero features locked

## Not addressed

Multi-user real-time collab. Classroom/instructor dashboards/grading. Curriculum sequencing. Per-user analytics. Per `NON-GOALS.md` w/ trigger.

## Tone

Direct. Terse. Accurate. No infantilizing, no gamification, no "great job!" toasts. Pedagogy is product.

No institution/course/exam refs in product surface. Domain language only — instruction, control signal, prime implicant, datapath.
